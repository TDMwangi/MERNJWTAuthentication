# MERN JWT Authentication

MERN authentication that stores a JWT in a HTTP cookie with Redux Toolkit for state management.

## _Start the server:_

```sh
npm run server
http://localhost:5000/api/users
```

## _Start the client:_

```sh
npm run client
http://localhost:3000
```

## _To run both the server and client concurrently:_

```sh
npm run dev
```
